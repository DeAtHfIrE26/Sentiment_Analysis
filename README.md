## 🌟 **Project Overview**

**KARY** is a cutting-edge sentiment analysis chatbot developed as part of an internship project. It leverages advanced AI and ML techniques to understand and respond to user emotions in **real-time**.

## 💻 **Installation & Setup**

```bash
# Clone the repository
git clone https://github.com/DeAtHfIrE26/Sentiment_Analysis.git

# Navigate to the project directory
cd Sentiment_Analysis

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py

Visit http://localhost:5000 in your browser to interact with KARY.

```

📊 Project Architecture
<div align="center">
</div>


    Input Processing: Advanced NLP techniques for text and speech input
    Sentiment Analysis: Deep learning models for emotion classification
    Context Engine: BERT-based models for understanding conversation context
    Response Generator: Fine-tuned language models for human-like responses
    Output Synthesis: Text-to-speech for voice responses
